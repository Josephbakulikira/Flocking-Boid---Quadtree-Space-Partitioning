# Flocking-Boid---Quadtree-Space-Partitioning

[Youtube](https://www.youtube.com/c/Auctux)

### Requirements
> Pygame

### Controls
        ESC                -> Close Window(quit)
        SPACE              -> Activate and Deactivate the Quadtree
        RETURN(ENTER) or q -> Draw(Render) the Quadtree
        r                  -> Show Query range
